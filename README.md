# 🧠 Sistema de Gerenciamento de Usuários

Projeto desenvolvido em Python com foco em prática de conceitos de backend, organização de código e manipulação de dados.

---

## 🚀 Funcionalidades

* ✅ Criar usuário
* 📋 Listar usuários
* 🔍 Buscar usuário por ID
* ❌ Remover usuário

---

## 🧠 Conceitos aplicados

* Separação de responsabilidades (services, utils, database)
* Validação de dados
* Manipulação de JSON
* Estrutura de projeto backend
* Organização em camadas

---

## 📂 Estrutura do projeto

* `main.py` → Interface com o usuário
* `servico_usuario.py` → Lógica de negócio
* `database.py` → Persistência de dados (JSON)
* `utilidades.py` → Funções auxiliares e validações

---

## ▶️ Como executar

```bash
python main.py
```
